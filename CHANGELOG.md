## [1.1.0] - 2025-11-29

### Added

- New IshChat Models:
  - Grok 3 Mini
  - Grok 4 Fast Reasoning
  - Grok 4 Fast Non-Reasoning
  - GPT OSS 120B
- New Image Models:
  - DeepImg (AI Image Generator)
  - Anime (Flux-1-Dev)

## [1.0.0] - 2025-11-28

### Added

- Initial release of Clave App
- Multi-model support with:
  - GPT-4o mini (Fast)
  - ChatGPT 3.5 (Lightweight)
  - Default (GPT-4o)
- Image generation capabilities
- Elegant UI/UX with dark mode
- Comprehensive API with documentation
- Security features including input validation and XSS protection
